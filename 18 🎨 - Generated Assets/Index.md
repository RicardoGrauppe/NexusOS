# Generated Assets

Repositório de tudo que foi gerado pelo Claude para esta empresa.

Cada arquivo aqui é concreto, reutilizável e versionado.

## Subpastas

| Pasta | Conteúdo |
|---|---|
| `Dashboards/` | Dashboards interativos de métricas e KPIs |
| `Reports/` | Relatórios completos em markdown |
| `HTML/` | Arquivos HTML standalone (abrir no navegador) |
| `Documents/` | Documentos gerados: propostas, SOPs formatados, apresentações |
| `Archive/` | Versões anteriores de qualquer asset acima |

## Convenção de nomenclatura

```
YYYY-MM-DD — [Tipo] — [Descrição].extensão
```

**Exemplos:**
- `2026-06-18 — Dashboard — Financeiro Q1 2026.html`
- `2026-06-18 — Report — Análise de Vendas Maio 2026.md`
- `2026-06-18 — Document — Proposta Cliente XYZ.md`

## Como solicitar um asset ao Claude

```
[GERAR] Dashboard HTML dos últimos 3 meses de finanças
[GERAR] Relatório de performance de vendas do Q2 2026
[GERAR] Documento de proposta para o cliente [Nome]
```
