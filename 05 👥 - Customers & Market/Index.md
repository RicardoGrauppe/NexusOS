## Jornada do Cliente

| Etapa | O que o cliente pensa / sente | Nossa ação |
|---|---|---|
| Consciência | `ex: "Tenho um problema mas não sei a solução"` | `ex: Conteúdo educativo` |
| Consideração | `ex: "Estou avaliando opções"` | `ex: Cases + demonstração` |
| Decisão | `ex: "Quero comprar, mas preciso de garantias"` | `ex: Trial gratuito + proposta` |
| Retenção | `ex: "Estou usando, mas posso churnar"` | `ex: Onboarding + check-ins` |
| Advocacy | `ex: "Estou satisfeito, indico para outros"` | `ex: Programa de indicação` |
